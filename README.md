# ASPATH dev site

パーキンソン病専門トレーニングスタジオ ASPATH（アスパス／鹿児島）の開発用サイト。

- 各ページは単一HTML（インラインCSS/JS）。
- 画像は `images/` に外部ファイルとして格納（WebP最適化・命名規則: `logo-` `hero-` `index-` `about-` `access-` `column-pk-` `icon-` `staff-`）。
- Webフォントは Google Fonts をCDN読み込み。
